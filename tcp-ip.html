Transmission Control Protocol / Internet Protocol

Definition

TCP/IP is a set of communication protocols used to connect devices on the Internet.

Explanation

TCP breaks data into packets, while IP ensures they are sent to the correct destination.

Example

Sending an email or also loading a website.
